# count100
Bài tập Nhóm 12
## Hướng dẫn
*Để build được cần cài Node >= 13.12 và npm >= 6.14*

```bash
git clone https://github.com/umuluto/count100.git
cd count100
```
Cài webpack, babel:
```bash
npm install
```
Chạy thử:
```bash
npm start
```
rồi vào trình duyệt mở localhost:3000

Build (khi hoàn thành dự án):
```bash
npm run build
```
